# Voice Persona Intake 검토 의견 ver1

신금자 작성본(2026-07-03)을 실데이터(통화 녹음 13건)·제품 기능명세와 대조한 검토 결과. 갱신 1건은 AI팀이 직접 반영해 파일로 첨부하고, 갱신 요청 2건과 설문지 v1.1 제안 3건을 정리했다.

---

## 1. [갱신 완료 — 파일 첨부] 원본 통화 녹음 보유 현황 정정

11-1 자료 보유 현황에 "원본 통화 녹음: 확인 필요 / 대상자 음성 파일 미포함"으로 기재돼 있었는데, **실제로는 통화 녹음 13건(약 2시간 분량, 대상자 목소리 포함)을 보유 중**이다 (AI팀 보관, 전사 완료). 이 항목은 AI팀이 직접 정정한 **갱신본 docx를 함께 보낸다** — 11-1의 녹음 보유 상태만 수정했고 나머지는 원본 그대로다.

## 2. [갱신 요청] 가족 지도에 인물 4명 추가

11-2 이름 리스트에는 있는데 3번 가족관계 지도에는 없는 인물이 4명 있다: **홍윤현, 남해욱, 신희용, 신욱**. 관계 정의가 없으면 AI 분석에서 전부 "누구인지 미확정"으로 분류된다. 관계는 가족만 알 수 있는 정보라 가족 지도에 행 추가를 부탁한다.

## 3. [갱신 요청] 호칭 미확정 항목 — 추정이라도 채워주기

아래 항목들이 "확인 필요"로 비어 있는데, 호칭은 화자 식별·지칭 해소("걔"가 누군지)의 핵심 재료다. **정답이 확실하지 않아도 가장 맞다고 생각하는 호칭으로 채워달라** (설문 원칙대로 "추정" 표시와 함께). 가족 검수 때 다시 확인하면 된다.

- 두 딸(이신화·이신경)을 부른 호칭
- 며느리 **성함 미기재** + 호칭
- 사위들(윤보영·남석찬) 호칭
- 이신화/이신리 호적명

## 4. [제안] 11-2 문항에서 인명 제외

지금은 같은 이름을 가족 지도와 11-2 두 곳에 중복 작성하게 돼 있어 위 2번 같은 불일치가 생긴다. **인명은 가족 지도에서만 수집하고 11-2에서는 아예 빼는 것**을 제안한다. 11-2는 "지명·음식 등 이름 외 고유명사"만 적는 문항으로 바꾸면 된다.

## 5. [제안] 5-2 "가족을 부를 때 쓰던 호칭" 문항 삭제

같은 정보를 3번 가족 지도의 "부른 호칭" 칸에서 이미 받고 있으니 **5-2의 이 문항은 빼면 된다**. (5-2의 "전화 받을 때 첫마디" 문항은 호칭과 별개 정보라 그대로 유지)

## 6. [제안] 11-2 "대표 음성 샘플(파일명·구간)" 문항 삭제

기능명세 VPA-005가 이 정보를 설문이 아니라 앱 화면(녹음 재생하며 시작/끝 구간 선택)에서 받도록 정하고 있다. 설문 시점에는 파일이 업로드 전이라 적을 수도 없다.
